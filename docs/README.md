# Project Documentation

## Directory Structure

- **`src/`**: Source code for the application.
  - **`app/`**: Next.js App Router pages and layouts.
  - **`components/`**: Reusable React components.
  - **`lib/`**: Utility functions and configurations.
  - **`assets/`**: Imported assets (images, icons) used in components.
- **`public/`**: Static assets served directly (e.g., favicon, robots.txt).
- **`scripts/`**: Utility scripts (e.g., audio processing).
- **`tests/`**: Test files.
- **`config/`**: Project-level configurations (non-framework).
- **`docs/`**: Project documentation.
